# 0x03. Shell, init files, variables and expansions

## Description
This project covers the basics of shell variables, environment management, and how initialization files work in Linux. It includes:

- Creating and managing local and global variables
- Modifying `$PATH`
- Working with aliases
- Using arithmetic and expansions
- Printing and formatting output

## File Descriptions
- `0-alias`: Creates an alias `ls` with value `rm *`
- `1-hello_you`: Prints `hello` followed by the current user
- `2-path`: Adds `/action` to `PATH`
- `3-paths`: Counts number of directories in `PATH`
- ...
